FAQ

How long is each pomodoro?
By default each work interval is 25 minutes followed by a 5-minute break.
Can I pause the timer?
Yes, press start again to resume where you left off.
Does it make a sound?
It plays a short beep when the session ends to remind you to stop.
Can I use it offline?
Yes, after loading once, the timer works without internet.
Is it really free?
Absolutely. Use the Pomodoro Timer as much as you want.
