FAQ

Can I export to file?
This simple demo doesn't save files, but you can copy the HTML to paste elsewhere.
Is syntax highlighting supported?
No highlighting, just live preview of the rendered Markdown.
Can I use keyboard shortcuts?
Standard text area shortcuts like copy, paste and undo work as normal.
Will large documents slow it down?
Performance is fine for typical posts up to a few thousand words.
Does it handle tables?
Yes, the preview uses the Marked library which supports GitHub style tables.
